\## ✨ Overview



\*\*CIWA Auto SMS\*\* is a professional WordPress plugin that automates SMS communication with your customers.



Instead of manually sending follow-up messages, the plugin automatically sends personalized SMS messages based on customer activities such as registration, purchases, order status changes, abandoned carts, and scheduled marketing campaigns.

\### 🧩 Smart Variables



Personalize every SMS automatically.



Supported placeholders:



| Variable | Description |

|-----------|-------------|

| `{first\_name}` | Customer first name |

| `{last\_name}` | Customer last name |

| `{full\_name}` | Full name |

| `{phone}` | Mobile number |

| `{email}` | Email |

| `{order\_id}` | Order ID |

| `{order\_total}` | Order Total |

| `{site\_name}` | Website Name |

| `{site\_url}` | Website URL |



Example:



```text

Hello {first\_name},



Your order #{order\_id} has been successfully registered.



Thank you for shopping at {site\_name}.

```
